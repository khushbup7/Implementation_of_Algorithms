Files:

ArrayedStack.java : Implements a stack using array
To create a stack : new ArrayedStack<>(Class<T[]> cls,size);
	cls is the class of the Objects to be stored on the stack.

TestArrayedStack.java : Test class to test the Arrayed stack

ListSetOperations.java : Implements union, intersection, difference of linkedlist implementing sorted sets.
To use the methods : ListSetOperations.intersect(List<T> l1, List<T> l2, List<T> outList)
		     ListSetOperations.union(List<T> l1, List<T> l2, List<T> outList)
         	     ListSetOperations.difference(List<T> l1, List<T> l2, List<T> outList)

TestList.java : Test class to test the ListSetOperations class

ShuntingYard.java : Implements the Shunting Yard  algorithm to convert infix expression to postfix expression

Token.java : Token class represents a element in the expression to be processed by Shunting Yard algorithm

Tokenizer.java : Class to convert the input string into a stream of Tokens to be processed by Shunting Yard algorithm
